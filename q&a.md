1. Difference between getElementById, getElementsByClassName, and querySelector(All)

getElementById selects one element by ID, getElementsByClassName selects multiple elements by class, and querySelector(All) selects elements using CSS selectors (first or all matches).

2. How do you create and insert a new element into the DOM?

You create an element using createElement() and insert it into the page using methods like append() or appendChild().

3. What is Event Bubbling?

Event bubbling means an event starts on the clicked element and then moves upward through its parent elements.

4. What is Event Delegation? Why is it useful?

Event delegation is handling events on a parent element instead of each child, which improves performance and works for dynamic elements.

5. Difference between preventDefault() and stopPropagation()?

preventDefault() stops the browser’s default action, while stopPropagation() stops the event from bubbling to parent elements.
